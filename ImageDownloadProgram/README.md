## 기능정의
- 각 요소에 해당하는 버튼을 클릭하면 사진을 다운로드 합니다.
- 하단 버튼을 클릭하면 모든 사진을 동시에 다운로드 합니다.
- 다운로드 진행상황은 progressView를 통해 나타냅니다.
